3. Runtime Attribution Control Exemption:
   Because these datasets carry zero copyright encumbrances, copyleft stipulations, or compulsory attribution requirements, the runtime mapping interface in this workload explicitly invokes Leaflet map creation with `attributionControl: false`. This zero-attribution, copyright-free configuration legally justifies rendering clean visual map reflow measurements during Speedometer benchmarking without introducing UI DOM overhead, external watermark rendering, or third-party copyright disclaimers.
